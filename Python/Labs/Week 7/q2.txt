Start the Python interpreter in interactive mode (with the python command) enter the following expressions and note what they evaluate to:

"Python" [ 0 ]
Ans = P

"Python" [ 1 ]
Ans = y

"Python" [- 1 ]
Ans = n

"Python" [- 2 ]
Ans = o

"Python" [ 1 : 3 ]
Ans = yt

"Python" [:- 1 ]
Ans = Pytho

What do negative indices do? 
Count backwards from the end of the string.

What do two numbers separated by a colon do?
Take a slice of the sequence, [left:right] left being the starting index (inclusive) and right the ending index (exclusive).
